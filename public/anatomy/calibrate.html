Anatomy SVG Path Calibration

Pick a vein → drag-draw on the image → repeat → click Export → paste into Claude.
How this works

Each vein you draw gets converted to an SVG <path d="…"> string in the 600×1453 viewBox of le-anterior.svg / le-posterior.svg. The exported JSON is fed into scripts/apply-calibration-json.mjs which rewrites the path coords in those SVG files.

Drag-drawing produces a polyline (M x,y L … L …). Hold Shift while drawing for a smoothed curve.
